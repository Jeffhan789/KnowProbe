# Security Policy

Security fixes target the latest release and the `main` branch.

Report undisclosed vulnerabilities through GitHub's **Report a vulnerability** flow in the repository Security tab, not a public issue. Include the affected version, reproduction steps, impact, and mitigation ideas. An acknowledgement is expected within 7 days.

Do not include live provider keys, proprietary prompts, private datasets, or sensitive model outputs. KnowProbe runs third-party model and retrieval integrations; users remain responsible for provider data-handling terms and for isolating untrusted experiment inputs.
